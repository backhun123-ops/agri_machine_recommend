# 🌾 농기계 길잡이

작물과 오늘 할 작업만 고르면 **필요한 농기계 → 사용·안전 안내 → 장비를 보유한 가까운 임대사업소 전화 연결**까지 한 번에 이어주는 웹 서비스입니다.

농기계 임대사업소는 전국 지자체가 운영하지만, "내 작업에 어떤 기계가 필요한지"와 "그 기계를 어느 임대소가 갖고 있는지"를 농민이 직접 알아내기 어렵다는 문제에서 출발했습니다.

## 실행 방법

별도 설치 패키지나 API 키 없이 바로 실행됩니다.

```bash
node server.js
```

실행 후 브라우저에서 **http://localhost:4173** 접속

- 요구 사항: Node.js 18 이상 (내장 `fetch` 사용)
- 포트 변경: `PORT=8080 node server.js`

## 주요 기능

1. **작물·작업 기반 농기계 추천** — 8개 작물 × 작업 단계(정식·방제·수확·건조 등)를 선택하면 규칙 기반 매칭 엔진이 적합한 농기계를 추천. 작물을 바꾸면 작업 목록이 자동으로 바뀜
2. **농민 눈높이 안내** — 추천 기계의 핵심 장비 세트, 작업 전 확인 사항, 사용 요령을 쉬운 말로 제공
3. **기계별 안전 수칙** — 트랙터 전도, 콤바인 막힘 제거, 방제 보호장비 등 기계 종류에 맞는 사고 예방 수칙 자동 표시
4. **임대소 매칭·비교** — 추천된 기계를 실제 보유한 임대사업소만 골라 거리·일일 임대료와 함께 최대 3곳 비교, 클릭으로 선택
5. **지도·경로·전화 연결** — 구글 지도 임베드로 위치·경로 표시(키 불필요), 버튼 한 번으로 임대소 전화 연결
6. **월별 농작업 안내** — 접속한 달 기준 주요 농작업 한 줄 안내
7. **큰 글씨 모드** — 고령 사용자를 위한 글자 확대 토글 (설정 저장)
8. **임대 이용 절차·FAQ** — 예약 → 안전교육 → 수령 → 반납 절차와 자격·요금·고장 대응 안내
9. **농기계 도감** (`machines.html`) — 임대 가능한 전체 농기계를 카드로 둘러보는 페이지. 기계별 실물 사진(위키미디어 커먼즈, 자유 라이선스)·소개·사용 요령·안전 수칙과 보유 임대소 목록을 제공하며, 임대소를 누르면 주소·전화번호·임대료가 펼쳐짐. 기계 이름·작물 검색 지원. 사진 로드 실패 시 아이콘으로 자동 대체

## 공공데이터 출처

서버가 시작 시 아래 API를 호출해 임대소·보유 장비 데이터를 수집하고 1시간 캐시합니다. (공공데이터포털 data.go.kr)

| 출처 | 내용 |
|---|---|
| 대전광역시 농기계임대 정보 API | GetFarmMachineRentalService |
| 세종특별자치시 농기계임대 정보 API | GetFarmMachineRentalService |
| 충청북도 농기계임대 정보 API | GetFarmMachineRentalService |
| 충청남도 농기계임대 정보 API | GetFarmMachineRentalService |
| 괴산군 농기계임대 정보 API | GetFarmMachineRentalService |
| 보령시 농업기계임대 정보 API | 환경변수 `BORYEONG_RENTAL_URL` 설정 시 사용 |
| 한국농촌경제연구원(KREI) 임대사업소 API | 환경변수 또는 요청 시 URL 지정 (`/api/krei-rental-data`) |

- API 장애·미승인 시에도 동작하도록 `data/rental-centers.json`에 대전·세종권 임대소 샘플 데이터를 내장 (폴백)
- 공공데이터 인증키는 `server.js`에 기본값이 있으며 `DATA_GO_KR_API_KEY` 환경변수로 교체 가능
- 임대료·보유 장비는 변동될 수 있어 화면에 "방문 전 전화 확인" 고지 표시

## 기술 스택

- **프런트엔드**: 순수 HTML/CSS/JavaScript (프레임워크 없음), Pretendard 웹폰트
- **백엔드**: Node.js 내장 `http` 모듈만 사용 (외부 의존성 0개, `npm install` 불필요)
- **지도**: Google Maps embed (API 키 불필요) / TMAP 경로 API 프록시(`/api/tmap-route`) 내장

## 파일 구조

```
agri_machine_software/
├── index.html               # 맞춤 추천 페이지
├── machines.html            # 농기계 도감 페이지
├── styles.css               # 공통 스타일 (반응형·큰글씨 모드 포함)
├── app.js                   # 추천 엔진·기계 데이터·임대소 매칭
├── catalog.js               # 도감 페이지 렌더링 (app.js 데이터 재사용)
├── server.js                # 정적 서빙 + 공공데이터 수집/캐시 API
└── data/rental-centers.json # 임대소 폴백 데이터
```

## API 엔드포인트

| 경로 | 설명 |
|---|---|
| `GET /api/rental-centers` | 내장 데이터 + 공공데이터 통합 임대소 목록 |
| `GET /api/config` | 서버 설정 상태 (키 등록 여부) |
| `POST /api/krei-rental-data` | KREI 사업소·보유기계 API 호출·정규화 |
| `POST /api/tmap-route` | TMAP 자동차 경로 탐색 프록시 |
